# labpyth3
# Генератор случайных рецептов

## Описание
Приложение **«Генератор случайных рецептов»** — это учебный проект, реализованный в рамках лабораторной работы №3   
Проект выполнен с применением **ООП**, поддерживает **GUI (PySide6)** и **веб-интерфейс (FastAPI)**, использует **SQLite** для хранения данных и логирование действий пользователя.

Функциональность:
- Добавление, редактирование и удаление рецептов  
- Генерация случайного рецепта (с фильтром по тегу)  
- Ведение журнала действий пользователя  
- Визуализация активности (график количества рецептов по датам)  
- Веб-версия с тем же функционалом  
- Тестовое покрытие >80% с использованием `pytest`

## Инструкции по запуску

### 1. Установка и запуск GUI-версии
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите графическое приложение:
   ```bash
   python -m app.main
   ```
3. Откроется окно программы с тремя вкладками:
   - **Рецепты** — список всех добавленных рецептов и график активности  
   - **Добавить** — форма добавления нового рецепта  
   - **Генератор** — выбор случайного рецепта

---

### 2. Запуск веб-версии
1. Перейдите в папку проекта и выполните:
   ```bash
   uvicorn web.main:app --reload
   ```
2. В браузере откройте:
   ```
   http://127.0.0.1:8000
   ```
3. На веб-странице можно:
   - Добавлять рецепты  
   - Просматривать список рецептов  
   - Генерировать случайный рецепт  
   - Смотреть график активности

## Краткая справка

| Раздел | Описание |
|:--|:--|
| **GUI** | Разработан с помощью **PySide6**. Содержит таблицу рецептов, логирование, вкладки для добавления и генерации. |
| **База данных** | Используется **SQLite**. Все данные хранятся в файле `recipes.db`. |
| **Тестирование** | Покрытие тестами (`pytest + pytest-cov`) более 80%. Проверяются модели, контроллеры и интеграционные сценарии. |
| **График активности** | Отображает количество добавленных рецептов по датам. Реализован с использованием **matplotlib**. |
| **Логирование** | Все действия пользователя отображаются в окне и сохраняются в консоли. |
| **Веб-версия** | Основана на **FastAPI**. Интерфейс на чистом HTML + Chart.js для графика. |
